# HyHub Frontend Starter (Angular 19)

This folder is a starter template for a dynamic, specification-driven UI.

## Commands

- Install: `npm install`
- Start: `npm start`
- Build: `npm run build`

## Scope

- Dynamic facility specification fetch
- Dynamic reactive form rendering
- API integration service placeholders

## API base URL

Configure `src/app/core/services/specification-api.service.ts` if backend host changes.
